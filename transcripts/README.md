# Transcripts

## Purpose

В этой папке хранятся исходные транскрипты разговоров с заказчиком, интервью, discovery calls и другие первичные материалы.

## Rules

1. Исходный транскрипт не должен изменяться после загрузки.
2. Если нужно очистить или структурировать транскрипт, создавать отдельный файл.
3. Не добавлять персональные данные без необходимости.
4. Не добавлять секреты, пароли, токены или credentials.
5. Каждый транскрипт должен иметь дату и понятное название.

## Naming Convention
YYYY-MM-DD-customer-name-topic.txt

Пример:

2026-05-11-acme-discovery-call.txt
