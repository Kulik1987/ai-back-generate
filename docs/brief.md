## 1. Executive Summary

Заказчик рассматривает кастомную разработку внутренней HRTech-платформы для крупного предприятия. По описанию, целевой продукт похож на один из модулей Sber Pulse и должен помогать с развитием сотрудников через анализ роли, задач и релевантного обучения. В разговоре обсуждались несколько возможных направлений: персонализация внутренних и внешних курсов, генерация индивидуальных планов развития, оценка компетенций и контроль прогресса обучения. Также поднималась идея использовать данные из task tracker для сопоставления фактической работы сотрудника с должностной ролью. Отдельно обсуждались более чувствительные сценарии, связанные с мониторингом действий сотрудника, но собеседники сами отметили высокий репутационный риск такого подхода. Из разговора видно, что границы решения пока не зафиксированы: неясно, нужен ли заказчику полный комплекс модулей или более узкий MVP. Перед подготовкой предложения участники договорились сначала уточнить задачу через дополнительный созвон с сокращенным списком вопросов. Данных для финальных требований пока недостаточно.
